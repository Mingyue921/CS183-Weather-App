##### Weather App & web

## Project Description
This is a weather application combined with AI and cultural content.
It provides weather information, AI suggestions, and traditional Chinese solar terms.

---

##  Features

- Real-time weather display
- AI suggestions (clothing, food, activities)
- 24 Solar Terms cultural cards
- Mobile-friendly UI

---

##  Tech Stack

- React Native (Expo)
- React.js (Web)
- Node.js (Backend)
- AI API (e.g., DeepSeek)

---

##  UI Design

Designed using Figma.
# Design Style
- Soft, calming, and minimal interface
- Low-saturation Morandi color palette
- Combination of modern UI and traditional Chinese elements
# Color System
- Soft white
- Light gray-blue
- Warm amber accents
